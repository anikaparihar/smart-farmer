# data/

Place the downloaded `crop_recommendation.csv` file here.

## Download Steps

1. Go to: https://www.kaggle.com/datasets/atharvaingle/crop-recommendation-dataset
2. Click **Download**
3. Rename the file to `crop_recommendation.csv`
4. Place it in this `data/` folder

## Dataset Columns

| Column | Description |
|--------|-------------|
| N      | Nitrogen content in soil (kg/ha) |
| P      | Phosphorus content in soil (kg/ha) |
| K      | Potassium content in soil (kg/ha) |
| ph     | pH value of the soil |
| label  | Crop recommended (target variable) |

The dataset contains 2200 rows across 22 crop types.
